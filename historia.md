# Història de Git

Git va ser creat l'any 2005 per Linus Torvalds, el mateix creador del nucli Linux. La seva creació va respondre a la necessitat de disposar d'un sistema de control de versions distribuït que fos ràpid, segur i flexible.

En aquell moment, la comunitat del kernel Linux necessitava un sistema que permetés gestionar de manera eficient els canvis en un projecte amb milers de col·laboradors. Git va néixer com una resposta a aquesta necessitat, oferint un model distribuït que permetia a cada desenvolupador tenir una còpia completa del repositori, millorant tant el rendiment com la seguretat.

Avui dia, Git és una eina essencial en el desenvolupament de programari. Serveix per rastrejar canvis en el codi, facilitar la col·laboració entre equips i gestionar versions de projectes de qualsevol mida. És utilitzat per empreses, organitzacions i desenvolupadors d'arreu del món, i plataformes com GitHub, GitLab i Bitbucket s'han construït al seu voltant per potenciar encara més el treball col·laboratiu.
